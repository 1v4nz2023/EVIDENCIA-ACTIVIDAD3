RELOJ DIGITAL

Para poder realizar este reloj, se utilizó las funciones de getMonth, getHours, getMinutes, getSeconds para poder leer los valores de la fecha y hora actual, se le aplicó un setInterval de 1 segundo para que se actualice la información mostrada en ese tiempo y se estableció una condicional para que se cambie de AM a PM según las doce.

CRONÓMETRO

Para poder realizar este cronómetro, utilizamos dos botones para empezar la cuenta y para ponerlo a 0, ambas funcionan bajo el evento "click" y se usaron dos funciones propias actualizar() que es un contador que va aumentando los milisegundos hasta aumentar a horas, la función iniciarContador() que hace que la cuenta se active y la función resetear() que envía a 0 la cuenta y se muestra en pantalla 0:0:0.

VIDEOJUEGO

Se crea un juego de memoria que nos muestra cartas las cuales debemos adivinar el par, tiene un contador de intentos y un contador de tiempo que cuando llegue a 0 pierdes o cuando los intentos sean 0, el codigo de este juego es un poco complicado, se genera una matriz con 6 cartas conteniendo imagenes, se aleatorian estas imagenes cada que recargue la página y empieza el juego.
